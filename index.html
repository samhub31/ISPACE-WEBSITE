Courses We Offer

GAME DEVELOPMENT

The art of creating games, the design, development and release of a game

APP DEVELOPMENT

In the App development course, students will learn the basics of coding Apps and games for smartphones.

ANIMATION

Animation is the process of giving the illusion of movement to drawings, models, or inanimate objects.

WEB DEVELOPMENT

Web development is a creative method and skill to create functionality in the web pages by using programming languages.

I.O.T (Internet of Things).

IoT is a technological concept in which the objects of daily life are equipped with electronic components, are connected to the internet and to each other.

ROBOTICS

Robotics is a hands-on and fun way for kids to learn STEM concepts by letting them make and write their own code for the robot built.

A.I/Machine Learning

The world is speedily evolving into the era of Machine Learning and A.I, and we see this as an opportunity to get kids trained so they grow and come up to speed with this technology.

3D Design

We train kids on 3d design , from using TinkerCAD to blender, we make sure your kid gets the appropriate training needed. .
